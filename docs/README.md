# PawapokeEventMaker

ノベルゲーム「pawapoke-MFver」で使用するイベント（キャラクターとセリフと画像が表示される一連の流れ）を作成・編集するWebアプリケーションです。

## 機能

- ✅ イベントの作成・編集・削除
- ✅ セリフの追加・編集・削除・順序変更
- ✅ イベントIDの自動生成
- ✅ ローカルストレージでのデータ保存
- ✅ dialogs.js形式へのエクスポート
- ✅ dialogs.js形式からのインポート

## 使い方

### 1. ブラウザで開く

`index.html` をブラウザで開いてください。

### 2. イベントの作成

1. 「新規イベント作成」ボタンをクリック
2. イベント名を入力（任意）
3. サムネイル画像のパスを入力（任意）
4. セリフを追加・編集

### 3. セリフの編集

- **テキスト**: セリフの内容を入力
- **話者**: 左または右を選択
- **画像パス**: キャラクター画像のパスを入力

### 4. データの保存

- 編集内容は自動的にローカルストレージに保存されます
- 「保存」ボタンで明示的に保存することもできます

### 5. エクスポート

1. 「エクスポート」ボタンをクリック
2. 表示されたコードをコピー
3. `dialogs.js` ファイルに貼り付け

### 6. インポート

1. 「インポート」ボタンをクリック
2. `dialogs.js` 形式のコードを貼り付け
3. 「確認」ボタンをクリック

## 開発

### ローカル開発

1. リポジトリをクローン
2. `index.html` をブラウザで開く
3. 開発者ツールでデバッグ

### ファイル構成

- **デザインシステム**: `css/design-system.css` にCSS変数を定義
- **コンポーネント**: `css/components.css` に再利用可能なコンポーネントを定義
- **サービス**: `js/services/` にビジネスロジックを実装
- **コンポーネント**: `js/components/` にUIコンポーネントを実装
